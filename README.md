Public Lighting Failure Prediction in Bogotá

Machine Learning project focused on predictive analysis of Bogotá’s public lighting system using CRISP-ML methodology and data engineering techniques to support preventive maintenance and smart city infrastructure management.

Technologies
- Python
- Flask
- Pandas
- Scikit-learn
- HTML/CSS
  
Dataset
Monthly public lighting records of Bogotá (2023), including:
- LED technology
- Sodium lamps
- Metal halide lamps
- Locality distribution
- Infrastructure statistics

  
models used
-XGBoost
-Logistic Regression
-Random Forest
-LightGBM
-K-Means Clustering
