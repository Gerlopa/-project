XGBoost Explanation

Machine Learning Model for Public Lighting Failure Risk Prediction

1. What is XGBoost?

XGBoost (Extreme Gradient Boosting) is an optimized gradient boosting algorithm known for its speed, scalability, and outstanding performance on structured tabular data. It has become one of the most widely used algorithms in data science competitions and real-world applications.

In this project, XGBoost classifies Bogotá's public lighting infrastructure into:

  • Low Failure Risk
  • Medium Failure Risk
  • High Failure Risk

2. How Does XGBoost Work?

XGBoost builds an ensemble of decision trees sequentially. Each new tree focuses on correcting the residual errors of the previous trees. It uses second-order gradient information (both gradient and Hessian) to optimize the loss function more precisely than standard gradient boosting.

Obj = Σ L(yᵢ, ŷᵢ) + Σ Ω(fₖ)

Where:

  • L(yᵢ, ŷᵢ) = Loss function measuring prediction error
  • Ω(fₖ) = Regularization term controlling tree complexity

3. Variables Used in the Model

Variable Description
% LED Percentage of LED infrastructure per locality
TOTAL Total public lighting points per locality

4. Key Features of XGBoost

  • Regularization: Built-in L1 and L2 regularization to prevent overfitting.
  • Second-order optimization: Uses both gradient and Hessian for more precise updates.
  • Parallel tree construction: Builds trees faster using parallelized computations.
  • Built-in cross-validation: Supports early stopping to find the optimal number of trees.
  • Sparsity awareness: Handles missing values natively.

5. Application in This Project

XGBoost is applied to classify lighting failure risk using LED adoption rates and total luminaire counts. Its regularization capabilities help prevent overfitting on the relatively small dataset of Bogotá's public lighting records.

The model helps identify:

  • High-risk localities requiring urgent maintenance
  • Zones with low LED adoption and elevated failure probability
  • Infrastructure patterns associated with high risk levels
  • Comparative risk assessment across Bogotá's localities

6. Advantages of XGBoost

  • Industry standard for structured tabular data
  • Built-in regularization reduces overfitting
  • Handles missing values automatically
  • Provides feature importance scores
  • Supports early stopping for optimal tree count

7. Model Limitations

  • More hyperparameters to tune than simpler models
  • Can overfit on very small datasets without regularization
  • Slower than LightGBM on large datasets
  • Risk labels are estimated rather than obtained from real failures

8. Conclusion

XGBoost provides a powerful and well-regularized approach for estimating public lighting failure risk. Its combination of gradient boosting with built-in regularization makes it particularly robust against overfitting, even with small datasets.

As one of the most trusted algorithms in applied machine learning, XGBoost delivers reliable risk classifications that support data-driven maintenance planning for Bogotá's public lighting system.
