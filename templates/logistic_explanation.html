Logistic Regression Explanation

Machine Learning Model for Public Lighting Failure Risk Prediction

1. What is Logistic Regression?

Logistic Regression is a supervised machine learning algorithm used for classification tasks. Its purpose is to predict the probability that a data sample belongs to a specific category.

In this project, Logistic Regression is used to classify Bogotá’s public lighting infrastructure into:

  • Low Failure Risk
  • Medium Failure Risk
  • High Failure Risk

2. How Does Logistic Regression Work?

The model analyzes relationships between infrastructure variables and predicts the probability of belonging to a specific risk category.

Logistic Regression transforms the prediction into probabilities using a mathematical function called the sigmoid function.

P(Y) = 1 / (1 + e-z)

Where:

  • P(Y) = Probability of a specific class
  • e = Euler’s number
  • z = Combination of the input variables

3. Variables Used in the Model

The model uses infrastructure-related variables extracted from Bogotá’s monthly public lighting records.

Variable Description
LED Number of LED light fixtures
Mh Metal halide light fixtures
Na Sodium light fixtures
TOTAL Total public lighting points
% LED Percentage of LED infrastructure

4. Application in This Project

The Logistic Regression model is applied to estimate potential failure risk in Bogotá’s public lighting infrastructure.

Since historical failure records are unavailable, the project estimates risk levels using infrastructure characteristics and technological distribution patterns.

The model helps identify:

  • Critical lighting zones
  • Areas with outdated infrastructure
  • Potential maintenance priorities
  • Technological modernization opportunities

5. Advantages of Logistic Regression

  • Easy to implement and interpret
  • Fast training process
  • Good baseline classification model
  • Efficient with structured datasets
  • Provides probabilistic predictions

6. Model Limitations

  • Performance may decrease with highly complex relationships
  • Requires sufficient data quality
  • Does not naturally capture nonlinear patterns
  • Risk labels are estimated rather than obtained from real failures

7. Conclusion

Logistic Regression provides a reliable and interpretable approach for estimating public lighting failure risk using infrastructure and technological indicators.

Although the project does not include real historical failure data, the model enables predictive infrastructure analysis and supports preventive maintenance decision-making strategies for Bogotá’s public lighting system.
